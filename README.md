# PDFer

A simple web application for working with PDF files.

## Features

- User registration and login
- PDF file handling
- History page
- Multilanguage support
- API folder for backend functionality
- Docker setup for development

## Tech Stack

- PHP
- Docker

## Project Structure

- `api/` — backend/API logic
- `docker/` — Docker configuration
- `lang/` — language files
- `login.php` — login page
- `register.php` — registration page
- `history.php` — user history
- `manual.php` — manual/help page
- `pdfile.php` — PDF-related functionality
